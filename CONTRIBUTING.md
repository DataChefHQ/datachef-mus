# Contributing to MUS

## Local development

```sh
git clone https://github.com/DataChefHQ/datachef-mus.git
cd datachef-mus
npm install
npm run dev       # starts the playground at localhost:5173
```

## Building

```sh
npm run build
```

## Submitting a PR

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Run `npm run build` and confirm it succeeds
4. Open a pull request — describe what you changed and why

## Reporting a bug

Use the GitHub issue tracker. Include steps to reproduce, expected vs actual behaviour, and your environment (framework, MUS version, browser).
